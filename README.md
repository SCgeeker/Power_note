# 统计检验力（功效）分析共同笔记

由中国Open Science讨论群发起，整理已知统计检定力分析方法及使用建议。

## 简易共同编辑
编辑者透过chrome或firefox，安装hackmd插件，即可在[hackmd平台](https://hackmd.io/)编辑，并推送更新。详细操作方法请见[官方操作文件](https://hackmd.io/c/tutorials/%2Fs%2Flive-hosting)。
